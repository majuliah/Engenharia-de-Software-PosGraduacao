# 02 - Interface do Usuário

## O designer e as entregas de UI

#### Designer é a pessoa criadora e design é o produto que ela cria. Ele entrega harmonia da tela como o alinhamento de itens, proporções dos elementos, estilos dos elementos, hierarquia das informações, legibilidade e diagramação das fontes. Ela entrega também apresentações, relatórios, protótipos, pesquisas, entregas para frontend. Basicamente é o cuidado em entregar a melhor experiência possível para o usuário.

## 02 -  As partes de uma interface digital

- #### Cabeçalho, miolo e rodapé. Em cada um pode haver:

  - #### menu, imagens, botões, campos, ícones, textos e links

  - #### Padronização, consistência visual: manter os elementos harmoniosos. São eles:

    - ### Cores, fontes, componentes, interação, consistência visual.

  #### Para praticar: visualizar elementos em telas e descreve-los

## 03 - Dispositivos e responsividade

#### São diferentes telas em que o produto será reproduzido. A responsividade é como uma tela responde a um dispositivo. São layouts:

- ### Layout fixo: Não muda, somente para um dispositivo específico.

- ### Layout fluído: Tamanho dos elementos definido por porcentagem. Se altera conforme o dispositivo

- ### Layout adaptativo: Layouts fixos pré definidos e o mais próximo ao tamanho da tela é exibido.

- ### Layout responsivo responde ao tamanho, resolução, orientação da tela

#### As grids ajudam a medir as resoluções para que seja mais fiel possível em um determinado dispositivo.

## 04 - O design system

#### Sistema de design é um sistema composto dos elementos que serão aplicados ao produto. São definidos de forma consistente em todo produto para criar a sensação de um projeto. As partes são tipografia, forma, cor e grid. Pode existir também padrão de texto, tom de voz, estratégia de conteúdo. Nos ajuda a economizar tempo e ter uma garantia de que um produto será padronizado, mantendo consistência e padrão visual.
